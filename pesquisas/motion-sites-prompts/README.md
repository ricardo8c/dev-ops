## 🎯 Resumo Executivo

**MotionSites.ai** é uma plataforma SaaS que oferece prompts premium prontos para criar landing pages animadas em minutos. Basta copiar, colar e publicar. A pesquisa encontra alternativas gratuitas e de baixo custo com bibliotecas de prompts semelhantes — cobrindo desde builders visuais (Framer, Dora, Wix ADI) até ferramentas de código (Replit + Gemini, Next.js + Cursor).

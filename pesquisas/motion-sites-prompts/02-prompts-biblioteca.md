## 🎨 Prompts de Animação CSS

#### Prompt 16 — Fade-in on Scroll
```
Add scroll-triggered animations to all sections:
- Elements start opacity: 0, translateY: 40px
- On scroll into view: animate to opacity: 1, translateY: 0
- Stagger delay: 100ms between elements
- Duration: 600ms, ease-out
- Trigger: IntersectionObserver at 20% threshold
Pure JS (no library).
```

#### Prompt 17 — Gradient Animation
```
Create a smooth animated gradient background:
- Colors: #667eea → #764ba2 → #f093fb → #667eea
- Animation: 15s linear infinite
- Overlay: semi-transparent gradient mesh
- Performance: use CSS animation, not JS
- Fallback: static gradient for reduced-motion
CSS only.
```

#### Prompt 18 — Hover Effects Collection
```
Add premium hover effects to all buttons and cards:
- Buttons: scale(1.05) + glow shadow on hover
- Cards: translateY(-8px) + shadow elevation
- Icons: rotate(15deg) + color shift
- Links: underline slide-in animation
- Images: subtle zoom + brightness
- All transitions: 300ms ease
Pure CSS.
```

## 📋 Como Usar Estes Prompts

1. **Replit**: Abra replit. com → New Project → Agent → Cole o prompt
2. **Framer**: Abra framer. com → New → Cole no AI prompt
3. **Dora**: Abra dora. design → Cole o prompt
4. **Cursor/Windsurf**: Abra → Cmd+K → Cole o prompt
5. **Manual**: Copie a estrutura → Cole em HTML/CSS

---

## ⚠️ Limitações Conhecidas

- Replit free tier: tempo limitado por projeto
- Gemini: pode hallucinar cores/tags
- Framer: algumas animações requerem plano pago
- Dora: ainda em beta, pode ter instabilidade
- Mobile: nem sempre testado em todos os devices

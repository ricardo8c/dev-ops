# 01 — Alternativas ao MotionSites.ai

## Plataformas e Ferramentas Semelhantes

---

### 🟢 Freemium / Grátis

#### Supahero.io
> *"A handpicked library of stunning website hero sections"*

Biblioteca curada de hero sections reais para inspiração e referência visual. Não gera código, mas fornece templates animados para estudar.

| Aspecto | Info |
|---------|-----|
| **Tipo** | Biblioteca de referência |
| **Prompts** | 100+ hero sections |
| **Preço** | Freemium |
| **Geração** | Não — visual reference only |
| **Melhor para** | Estudar design, criar mockups |
| **URL** | supahero. io |

#### Dora (dora.design)
Builder de sites 3D animados com navegação em perspectiva. Permite criar sites com efeitos imersivos em canvas 3D. Prompt-based design.

| Aspecto | Info |
|---------|-----|
| **Tipo** | Builder visual 3D |
| **Prompts** | Prompt-based AI |
| **Preço** | Grátis (beta) |
| **Features** | WebGL, 3D, câmera, parallax |
| **Export** | HTML embedável |
| **URL** | dora. design |

#### Replit + Gemini
Combinação gratuita: Replit Agent gera código completo a partir de prompts em linguagem natural. 10 prompts completos disponíveis para diferentes tipos de landing pages.

| Aspecto | Info |
|---------|-----|
| **Tipo** | Código + AI agent |
| **Prompts** | 10 prompts + custom |
| **Preço** | Freemium (Replit free tier) |
| **Tecnologia** | Next.js, React, Tailwind |
| **Export** | Código fonte completo |
| **URL** | replit. com |
| **Tutorial** | creatoreconomy. so |

#### Taskade Genesis
Gera websites completos e funcionais a partir de prompts em linguagem natural. Diferencial: sites evoluem com dados do workspace.

| Aspecto | Info |
|---------|-----|
| **Tipo** | AI website generator |
| **Prompts** | Natural language |
| **Preço** | Freemium |
| **Output** | HTML funcional |
| **URL** | taskade. com |

#### DesignRocket (designrocket. io)
Tutorial em vídeo com prompts exclusivos para criar landing pages animadas em 12 minutos. Usa Replit + Gemini.

| Aspecto | Info |
|---------|-----|
| **Tipo** | Tutorial + prompts |
| **Prompts** | Exclusivos (vídeo) |
| **Preço** | Freemium |
| **URL** | designrocket. io |

## Métricas Principais

- **Intelligence Index** (Artificial Analysis) — 0 a 60
- **Arena AI** — ranking comunitário
- **Velocidade** — tokens/segundo
- **Custo** — USD por 1M tokens
- **Benchmarks específicos** — SWE-bench, HLE, MMMU-Pro, etc.
